PROJECT DESCRIPTION

Industrial Warehouse

The industrial warehouse used for logistic purpose is built with the latest technology providing the highest possible sunlight using transparent RIDGE Ventilation system. The single PEB Structure with 51 feet height surrounded by green space along with 6 docking station makes it unique in its category.

PROJECT DETAIL

  • Location: Sape, Bhiwandi, Maharashtra, India
  • Type: Commercial
  • Total Developed Area: 75,000 Sq.Ft.
  • Period of Completion: 7 months
  • Completion Year: 2018
